# iRTSP - Go
Go implementation of the iRTSP protocol created by Ubitus

## Supported features
- [ ] TCP transport
- [ ] UDP transport
- [x] START messages
- [ ] SETUP messages
- [ ] KNOCK messages
- [ ] PUSHINFO messages
- [ ] USERINFO messages
- [ ] REMOTECTL messages
- [ ] SET messages
- [ ] PLAY messages
- [ ] ECHO messages
- [ ] TEARDOWN messages
